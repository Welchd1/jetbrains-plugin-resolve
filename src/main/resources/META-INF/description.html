RESOLVE Language support for IntelliJ IDEA

GitHub | Bug Tracker

This plugin turns your copy of IntelliJ IDEA into a Formalization Integrated Development Environment (F-IDE) for the RESOLVE specification and programming language. The plugin requires a local copy of RESOLVE to be installed. Links and download instructions can be found at the official website.
Features:
  • Advanced syntax highlighting for both programmatic and abstract specifications
  • for all built-in RESOLVE keywords and live templates to facilitate insertion of definitions
  • Refactor/rename for all named programmatic and mathematical entities (including modules)
  • Code folding support for all scoped programmatic and mathematical constructs
  • Quick navigation between symbol usages with Navigate - Related Symbol
  • Quick navigation to the declaration of a functions
  • Import of other Mathematica packages as library for completion and highlighting of external package functions
  • Goto Symbol for quick navigation to a function by typing its name
  • Spellcheck that already contains the dictionary of all Mathematica symbols
  • Code folding of expressions and sections
  • Live-Templates to create all frequently used constructs
  • Surround providers that can wrap an expression with another one
